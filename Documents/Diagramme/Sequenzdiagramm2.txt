title Control Software Startup and Connection to Server Sequence Diagram

User->Control Software: Start
Control Software->Network: Broadcast for server
    Network->Server: Search for server
alt Server Response
    Server->Network: Response with list of available MBots
    Network->Control Software: Receive list of available MBots
else Timeout
    Control Software->Control Software: Retry Broadcast after 5s
    loop Repeat
        Control Software->Network: Retry Broadcast
    end
end
Control Software->Network: Establish TCP connection
Network->Server: Establish TCP connection

alt Connection okay
    Server->Network: Confirm connection
    Network->Control Software: Confirm connection
else Timeout
    Control Software->Control Software: Retry connection after 5s
    loop Repeat
        Control Software->Network: Retry connection
    end
end
